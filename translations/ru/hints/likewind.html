Вы можете посчитать количество первого элемента и если это число совпадет с общим количеством элементов (длиной списка), значит все элементы списка действительно одинаковы. Также не забудьте учесть особый случай - пустой список, который, согласно условиям задачи, тоже должен возвращать True.
Вы могли бы указать больше подробностей?
Всё достаточно просто. Вам необходимо посчитать количество вхождений первого элемента в список. Например, используя метод list.count(x). Об этом и других методах для работы с list вы можете прочесть здесь. Он работает следующим образом: 
elements = ['a', 'a', 'c', 'd', 'e', 'e', 'a', 'b']
elements.count('a')
elements.count('a') --> 3
Затем нужно сравнить это число с длиной списка. Если эти значения равны - значит список состоит только из одинаковых елементов.
А если я знаю только то, что нужный элемент находится на первой позиции?
Если нам нужен элемент, находящийся на определенной позиции - можно обращаться по индексу:
elements.count(elements[0])
Позиция с индексом 0 - первая позиция в списке.
Отлично. А что делать с данными о количестве первого элемента?
Теперь вам необходимо сравнить эту информацию с длиной исходного списка (общим количеством элементов). Если мы вернемся к первому примеру: 
elements = ['a', 'a', 'c', 'd', 'e', 'e', 'a', 'b']
elements.count(elements[0])
и узнаем, равно ли количество первого элемента общему количеству элементов в списке, то узнаем, состоит ли список из всех одинаковых элементов или нет.
elements.count('a') == len(elements) --> False, так как количество первого элемента = 3, а всего элементов в списке 8. В другой ситуации: 
elements = ['a', 'a', 'a', 'a']
elements.count(elements[0])
elements.count(elements[0]) == len(elements) --> True, так как эти значения (количество первого элемента и количество элементов в списке вообще - совпадают)
И как мне описать это в коде?
Если сложить всё вместе, мы получим: 
def all_the_same(elements):
    if len(elements) <= 1:
        return True
    if elements.count(elements[0]) == len(elements):
        return True
    return False
Этот же код можно переписать более элегантным образом: 
def all_the_same(elements):
    if elements == []:
        return True
    return elements.count(elements[0]) == len(elements)
